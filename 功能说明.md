# 陪伴系统功能升级指南

## 新增功能概览

本次升级为陪伴系统增加了以下核心功能：

### 1️⃣ 形象自定义
**位置**：形象页面 → 🎨 自定义形象

- **上传照片**：用户可以上传自己的照片（支持 JPG、PNG 格式）
- **动画形象**：系统会生成用户的自定义动画形象
- **实时预览**：上传前可预览效果
- **持久存储**：自定义形象会被保存到本地存储

**操作步骤**：
1. 进入形象页面
2. 点击"🎨 自定义形象"按钮
3. 选择你的照片
4. 预览效果后点击"✓ 确认使用"

---

### 2️⃣ 对话窗口
**位置**：形象页面 → 💬 与形象对话

- **实时对话**：与你的陪伴精灵进行自由对话
- **智能回复**：AI 会根据上下文生成合适的回复
- **聊天历史**：对话内容会在窗口中显示
- **内容检测**：系统会自动检测并拒绝不当内容

**操作步骤**：
1. 进入形象页面
2. 在对话框中输入你想说的话
3. 点击"发送"按钮或按 Enter
4. 形象会给出回复

---

### 3️⃣ 语句训练
**位置**：形象页面 → 🗣️ 训练形象语句

- **自定义语句**：教会形象说你喜欢的话
- **学习应用**：形象会在适当时机使用这些语句
- **安全防护**：系统不允许训练含有违法、暴力内容的语句
- **管理列表**：可以查看、删除已训练的语句

**操作步骤**：
1. 进入形象页面
2. 点击"🗣️ 训练形象语句"按钮
3. 输入要训练的语句（最多 200 字）
4. 点击"🚀 开始训练"
5. 已训练的语句会显示在列表中

**安全说明**：
⚠️ **禁止内容**：
- 暴力、恐怖内容
- 色情内容
- 赌博、毒品相关
- 诈骗、犯罪相关
- 其他违法乱纪内容

系统会自动检测并阻止这类内容，确保形象的言论文明健康。

---

### 4️⃣ 任务界面颜色自定义
**位置**：个人中心 → 任务界面主题色

- **颜色选择**：使用颜色选择器选择喜欢的颜色
- **即时预览**：显示当前选中的颜色代码
- **全局应用**：主题色会应用到所有按钮和界面元素
- **持久保存**：颜色设置会被保存

**操作步骤**：
1. 进入个人中心
2. 找到"任务界面主题色"选项
3. 点击色块打开颜色选择器
4. 选择你喜欢的颜色
5. 点击"应用主题色"按钮

---

### 5️⃣ 语音文件上传
**位置**：个人中心 → 形象声音文件

- **自定义语音**：上传你的语音文件作为形象的声音
- **支持格式**：MP3、WAV、M4A 格式
- **训练集成**：与自定义语句配合使用效果更佳
- **本地存储**：语音文件保存在本地

**操作步骤**：
1. 进入个人中心
2. 找到"形象声音文件"选项
3. 选择你的语音文件
4. 点击"上传声音"按钮
5. 上传成功后会有提示

---

## 安全与隐私

### 内容安全机制
- ✅ **自动检测**：系统会实时检测用户输入的内容
- ✅ **关键词过滤**：包含违法关键词的内容会被拒绝
- ✅ **友好提示**：用户会收到清晰的警告信息
- ✅ **不会记录**：违规内容不会被保存

### 数据存储
- 💾 所有用户自定义数据都保存在浏览器本地存储（localStorage）
- 🔒 不会上传到服务器（仅在调用 AI 时发送必要信息）
- 🗑️ 清除浏览器数据会清除所有保存的自定义设置

---

## 使用建议

### 最佳实践
1. **形象自定义**：上传清晰、正面的照片效果更好
2. **语句训练**：选择简短、易记的语句（5-20 字最佳）
3. **颜色搭配**：选择与任务页风格一致的颜色
4. **对话互动**：经常与形象对话可以提升互动等级

### 常见问题

**Q: 上传的照片会被保存吗？**
A: 只保存在你的浏览器本地，不会上传到任何服务器。

**Q: 可以训练多少个语句？**
A: 理论上无限制，但建议不超过 50 个，以保持形象回复的多样性。

**Q: 如何恢复默认设置？**
A: 清除浏览器的网站数据即可恢复所有默认设置。

**Q: 语音文件如何使用？**
A: 上传后系统会在合适时机播放（目前集成在试听和提醒中）。

## 技术细节

### 前端实现
- 使用 HTML5 File API 处理文件上传
- 使用 Canvas 或 Image 处理图片预览
- 使用 localStorage 实现数据持久化
- 使用 Web Speech API 实现语音播报

### 后端支持
- DeepSeek AI 提供智能对话能力
- 支持自定义语句集成
- 内置安全检查机制
- 提供通用的对话和任务提醒 API

### 安全架构
```
用户输入 → 前端检测 → 后端再检测 → AI 生成 → 安全回复
   ↓
  拒绝
```
